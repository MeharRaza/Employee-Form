# Simple CRUD Form Application

A responsive and beginner-friendly CRUD (Create, Read, Update, Delete) Form Application developed using HTML, CSS, and JavaScript. This project allows users to add, edit, delete, and manage records dynamically through a simple user interface.

---

## 🚀 Features

- Add new records
- Edit existing records
- Delete records
- Responsive user interface
- Real-time data handling using JavaScript
- Clean and simple design
- Beginner-friendly project structure

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
